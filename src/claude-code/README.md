# Claude Code

Installs Claude Code, Anthropic's official CLI for Claude. Requires Node.js 18+.

## Options

| Option | Description | Type | Default |
|---|---|---|---|
| `version` | Version of Claude Code to install (e.g., 'latest', '1.0.0') | string | `latest` |

## Usage

```json
"features": {
    "claude-code": {}
}
```
