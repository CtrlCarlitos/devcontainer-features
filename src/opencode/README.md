# OpenCode

Installs OpenCode, a powerful AI coding agent built for the terminal. Requires Node.js 18+.

## Options

| Option | Description | Type | Default |
|---|---|---|---|
| `version` | Version of OpenCode to install (e.g., 'latest', '1.0.0') | string | `latest` |

## Usage

```json
"features": {
    "opencode": {}
}
```
