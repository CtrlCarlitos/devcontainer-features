# Codex

Installs Codex, OpenAI's lightweight coding agent that runs in your terminal. Requires Node.js 22+.

## Options

| Option | Description | Type | Default |
|---|---|---|---|
| `version` | Version of Codex to install (e.g., 'latest', '1.0.0') | string | `latest` |

## Usage

```json
"features": {
    "codex": {}
}
```
