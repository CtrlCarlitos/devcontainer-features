# Node.js

Installs Node.js if not present. Validates existing installations meet minimum version requirements.

## Options

| Option | Description | Type | Default |
|---|---|---|---|
| `version` | Node.js version to install if not present (e.g., '22', '20', 'lts') | string | `22` |

## Usage

```json
"features": {
    "node": {}
}
```
