# Dev Container Features

This repository provides a small set of Dev Container Features for AI tooling and developer ergonomics.

## Features

- `node` (base dependency)
- `bmad-method`
- `claude-code`
- `codex`
- `opencode`
- `nerd-font`

## Supported Base Images

Debian/Ubuntu only (scripts rely on `apt-get`). Other distros are not supported yet.

## Local Usage

Add features to your `.devcontainer/devcontainer.json`:

```json
{
  "features": {
    "./src/node": { "version": "22" },
    "./src/claude-code": { "version": "latest" },
    "./src/codex": { "version": "latest" },
    "./src/opencode": { "version": "latest" }
  }
}
```

## Testing

Run scenario tests with the Dev Containers CLI:

```bash
devcontainer features test --skip-autogenerated --project-folder .
```

Scenarios use the local `node` feature to avoid registry dependencies.

## CI/CD

GitHub Actions workflows exist in `.github/workflows/` but are **pre-production**. Enable publishing only once features are production-ready.
